# 🏠 House Pricing Prediction Models

Proyecto desarrollado en el marco del **Curso de Inteligencia Atificicial de Samsumng Innovation Campus** por **@el-biomedico**, **@mmarzap19** y **@josecmcinco**.

## 📌 Descripción

Este proyecto tiene como objetivo analizar y predecir precios de viviendas utilizando distintos modelos de machine learning. A través de un enfoque práctico, se exploran técnicas de análisis de datos, preprocesamiento y modelado para obtener predicciones lo más precisas posibles.

El trabajo se centra en un flujo completo de ciencia de datos, desde la exploración inicial hasta la evaluación de modelos.

## 📈 Resultados

El proyecto muestra cómo diferentes enfoques de modelado afectan la precisión de las predicciones, permitiendo identificar qué técnicas funcionan mejor para este tipo de problema.

## 📑 Índice del proyecto

    Estudio de Datos

        1. Gestión de datos vacíos e imputación
        2. Estudio de valores anómalos
        3. Normalizar variables numéricas
        4. Primera versión
        5. Numeración de las variables categóricas
        6. Transformación de la etiqueta
        7. Partición de datos de entrenamiento
        8. Estudio y extracción de características


    Entrenamiento de modelos

        9. Regresión lineal
        10. Regresión lineal con Lasso
        11. Decision Tree
        12. Bagging - Random Forest
        13. ADA Boost
        14. Resultados de los entrenamientos
        15. Predicción con los modelos seleccionados usando los datos de test

# 🏠 House Pricing Prediction Models

Project developed during the **Artificial Intelligence Course at Samsung Innovation Campus** by **@el-biomedico**, **@mmarzap19** and **@josecmcinco**.

## 📌 Description

This project aims to analyze and predict housing prices using different machine learning models. Through a practical approach, various data analysis, preprocessing, and modeling techniques are explored in order to achieve the most accurate predictions possible.

The work follows a complete data science workflow, from initial exploration to model evaluation.

## 📈 Results

The project shows how different modeling approaches impact prediction accuracy, allowing the identification of the most effective techniques for this type of problem.

## 📑 Project Index

**Data Analysis**

1. Handling missing data and imputation  
2. Outlier analysis  
3. Normalization of numerical variables  
4. First version  
5. Encoding categorical variables  
6. Target transformation  
7. Training data split  
8. Feature analysis and extraction  

**Model Training**

9. Linear Regression  
10. Linear Regression with Lasso  
11. Decision Tree  
12. Bagging - Random Forest  
13. AdaBoost  
14. Training results  
15. Prediction using selected models on test data
